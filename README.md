# הבית שלנו

PWA פרטית לניהול והשוואת נכסים משותפת.

## הפעלה מקומית
1. העתק `.env.example` לקובץ `.env.local` והכנס מפתחות Supabase.
2. הרץ `npm install`.
3. הרץ `npm run dev`.

## Supabase
1. הרץ `supabase/schema.sql` ב-SQL Editor.
2. צור שני משתמשים ב-Authentication > Users.
3. עדכן אימיילים בתוך `supabase/create-family.sql` והרצה פעם אחת.
